Scientific Output

Publications

Computational methodologies rigorously validated and published in peer-reviewed ecological journals, adhering to the TRACE Framework for transparency and reproducibility.

Google Scholar ORCID Profile ResearchGate
My computational models are developed following the TRACE framework (Transparent and Comprehensive Ecological Modelling) and validated against independent empirical datasets. All first-author papers provide full reproducible code via GitHub repositories.
